# reverse_motorDirection

이 스케치는 누르는 버튼에 따라 DC 모터의 방향을 바꿔주는 스케치입니다. 

작은 모터는 그냥 사용할 수 있지만, 사용하는 모터의 사용전류량에 따라 외부전원을 인가하거나 트랜지스터를 사용할 필요가 있습니다. 

|아두이노 | 장치 |
| :-------: | :------: |
| Vin     | 배터리 + |
| GND     | 배터리 - |
| D2      | 버튼 1   |
| D3      | 버튼 2   | 
| D7      | 모터 + or -, led1 +, led2 -|
| D8      | 모터 - or +, led1 -, led2 +|
